Agente com ferramentas (busca web e calculadora)

ATENÇÃO: habilitar esta opção permite que o assistente decida sozinho fazer chamadas de rede externas (busca web) para responder perguntas, fora do funcionamento offline-first padrão do QuestResume. Só ative se confiar no serviço de busca configurado abaixo.

Webhooks e notificações

Cadastre uma URL para receber um POST com JSON sempre que um evento ocorrer: indexação concluída, erro em documento ou pergunta respondida.
